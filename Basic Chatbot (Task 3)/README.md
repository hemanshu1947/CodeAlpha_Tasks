# Basic ChatBot

A simple rule-based console chatbot built with Python. It replies to predefined user inputs such as greetings, name questions, thank-you messages, and exit commands.

## Features

- Rule-based responses using conditional statements
- Interactive command-line conversation loop
- Case-insensitive user input handling
- Exit command using `bye`

## Tech Stack

- Python 3
- Standard Python libraries only

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/basic-chatbot.git
cd basic-chatbot
```

2. Run the Python file:

```bash
python basic_chatbot.py
```
